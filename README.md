# 🎮 Play Store

A modern Play Store-inspired web application built with React, where users can browse trending apps, view app details, and install their favorite apps.

## 🌐 Live Demo

[Play Store Live](https://play-store-1.netlify.app/)

---

## 📸 Features

- 🔥 **Trending Apps** — Browse the most popular apps with download counts and ratings
- 📄 **App Details Page** — View detailed info about each app including company, reviews, and size
- 📦 **Install Apps** — Install apps and track your installed apps list
- 📱 **Responsive Design** — Fully responsive for mobile, tablet, and desktop
- ⚡ **Fast Loading** — Built with Vite for blazing fast development and build times

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React 19 | UI Framework |
| React Router 7 | Client-side routing |
| Tailwind CSS 4 | Styling |
| DaisyUI | UI Component library |
| React Icons | Icon library |
| React Spinners | Loading animations |
| React Toastify | Toast notifications |
| Recharts | Data visualization |
| Vite | Build tool |

## 🚀 Getting Started

### Installation

1. Clone the repository:

```bash
git clone https://github.com/mdshihabshahriar/Play-Store.git
```

2. Navigate to the project directory:

```bash
cd Play-Store
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit `http://localhost:5173`
